oSaint is a tool for Active OSINT (Open-Source Intelligence). It was made and tested on Linux.

# Features
• WHOIS Lookup  
• DNS Enumeration  
• Subdomain Brute Forcing  
• HTTP Fingerprinting (WhatWeb)  
• Aggressive Nmap Scan  
• Auto-installs missing tools (universal, no apt, no pacman needed)
• 100% portable script — works on ANY distro

# Installation
```
git clone https://github.com/jamaal-hussein/osaint/
cd osaint
sudo chmod +x oSaint
sudo mv oSaint /usr/bin/
sudo rm -rf ~/osaint
```

# Usage
```
oSaint [target.com]
```
